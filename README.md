# Projeto: Landing Page - Pacote +150 Moldes Infantis

## Sobre o Projeto
Este projeto visa criar uma landing page de alta conversão para a venda de um pacote de moldes infantis, modelada no estilo de sites de sucesso do nicho.

## Estrutura
- `index.html`: Página principal de vendas upon
- `styles.css`: Estilização customizada
- `assets/`: Imagens e ícones
